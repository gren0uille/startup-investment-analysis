# Исследование рынка стартапов для инвестиционных решений

## О проекте
Комплексный анализ динамики и структуры финансирования стартапов за 2000–2014 годы.
Цель — подготовить обоснованные рекомендации для инвесторов на 2015 год на основе
исторических данных о 40 000+ стартапах.

## Задачи
- Провести предобработку и очистку двух датасетов (54 294 строки)
- Сегментировать стартапы по срокам и типам финансирования
- Выявить растущие рыночные сегменты и динамику возвратов
- Сформулировать инвестиционные рекомендации на основе данных

## Ключевые результаты
- Очищено ~25% некачественных данных: устранено **4 855 дубликатов**,
  восстановлены пропуски в датах, нормализованы текстовые поля —
  итоговый датасет **40 907 записей**
- Выявлено, что стартапы с повторными инвестициями (срок > 1 года) составляют
  лишь **30% компаний**, но привлекают **62% объёма всех инвестиций**
- Определены наиболее перспективные сегменты: **Medical** (рост с 11 млн до 175 млн USD),
  **Internet** (рост до 117 млн USD), **Big Data** и **SaaS** — кратный рост в 2010–2014
- По типам финансирования наивысший возврат показали **Debt_financing (132%)** и
  **Private_equity (114%)** в 2014 году, Venture — восходящий тренд (81%)
- Обнаружена обратная зависимость: пик числа раундов в 2013 (12 320) при снижении
  медианного размера раунда — инвесторы стали распределять средства между
  большим числом небольших стартапов

## Рекомендации для инвестора (2015)
- **Сегменты:** Medical, Internet — стабильный рост; Big Data, SaaS — новые перспективные направления
- **Типы финансирования:** Debt_financing и Private_equity — наиболее высокий возврат;
  Venture — долгосрочная перспектива
